# Правильная установка Nginx для Windows

## Официальный сайт

**Официальный сайт:** https://nginx.org/en/download.html

## Скачивание

1. Перейдите на https://nginx.org/en/download.html
2. Выберите раздел **"Stable version"** (стабильная версия)
3. Скачайте файл **nginx/Windows-X.X.X** (например, `nginx/Windows-1.28.1`)
4. Распакуйте архив в удобное место (например, `C:\nginx`)

## Предупреждение о цифровой подписи

⚠️ **Это нормально!** Официальные бинарники nginx для Windows **не имеют** цифровой подписи Microsoft. Это не означает, что файл опасен или неправильный.

### Как обойти предупреждение:

**Вариант 1: Разрешить выполнение для текущей сессии**
```powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
```

**Вариант 2: Разрешить для конкретного файла**
```powershell
Unblock-File -Path "C:\Users\aliaksei\Downloads\nginx-1.28.1\nginx-1.28.1\nginx.exe"
```

**Вариант 3: Запуск через cmd (не PowerShell)**
```cmd
cd C:\Users\aliaksei\Downloads\nginx-1.28.1\nginx-1.28.1
nginx.exe
```

## Проверка подлинности

Если хотите убедиться, что скачали правильный файл:

1. Проверьте размер файла (должен быть около 1-2 МБ)
2. Проверьте, что скачали с официального сайта nginx.org
3. Не скачивайте с других сайтов или торрентов

## Альтернатива: Использование через Chocolatey

Если у вас установлен Chocolatey:

```powershell
choco install nginx
```

Это установит nginx в стандартное место и настроит службу.

## Текущая установка

Ваш nginx находится в: `C:\Users\aliaksei\Downloads\nginx-1.28.1\nginx-1.28.1\`

Это нормальное расположение, если вы распаковали архив из Downloads. Можно оставить так или переместить в `C:\nginx` для удобства.
